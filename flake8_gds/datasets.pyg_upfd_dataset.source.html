22 class PyGUPFDDataset(InMemoryDataset):
23 r"""The tree-structured fake news propagation graph classification dataset
24 from the `"User Preference-aware Fake News Detection"
25 <https://arxiv.org/abs/2104.12259>`_ paper.
26 It includes two sets of tree-structured fake & real news propagation graphs
27 extracted from Twitter.
28 For a single graph, the root node represents the source news, and leaf
29 nodes represent Twitter users who retweeted the same root news.
30 A user node has an edge to the news node if and only if the user retweeted
31 the root news directly.
32 Two user nodes have an edge if and only if one user retweeted the root news
33 from the other user.
34 Four different node features are encoded using different encoders.
35 Please refer to `GNN-FakeNews
36 <https://github.com/safe-graph/GNN-FakeNews>`_ repo for more details.
37  
38 .. note::
39  
40 For an example of using UPFD, see `examples/upfd.py
41 <https://github.com/rusty1s/pytorch_geometric/blob/master/examples/
42 upfd.py>`_.
43  
44 Args:
45 root (string): Root directory where the dataset should be saved.
46 name (string): The name of the graph set (:obj:`"politifact"`,
47 :obj:`"gossipcop"`).
48 feature (string): The node feature type (:obj:`"profile"`,
49 :obj:`"spacy"`, :obj:`"bert"`, :obj:`"content"`).
50 If set to :obj:`"profile"`, the 10-dimensional node feature
51 is composed of ten Twitter user profile attributes.
52 If set to :obj:`"spacy"`, the 300-dimensional node feature is
53 composed of Twitter user historical tweets encoded by
54 the `spaCy word2vec encoder
55 <https://spacy.io/models/en#en_core_web_lg>`_.
56 If set to :obj:`"bert"`, the 768-dimensional node feature is
57 composed of Twitter user historical tweets encoded by the
58 `bert-as-service <https://github.com/hanxiao/bert-as-service>`_.
59 If set to :obj:`"content"`, the 310-dimensional node feature is
60 composed of a 300-dimensional "spacy" vector plus a
61 10-dimensional "profile" vector.
62 split (string, optional): If :obj:`"train"`, loads the training
63 dataset.
64 If :obj:`"val"`, loads the validation dataset.
65 If :obj:`"test"`, loads the test dataset.
66 (default: :obj:`"train"`)
67 transform (callable, optional): A function/transform that takes in an
68 :obj:`torch_geometric.data.Data` object and returns a transformed
69 version. The data object will be transformed before every access.
70 (default: :obj:`None`)
71 pre_transform (callable, optional): A function/transform that takes in
72 an :obj:`torch_geometric.data.Data` object and returns a
73 transformed version. The data object will be transformed before
74 being saved to disk. (default: :obj:`None`)
75 pre_filter (callable, optional): A function that takes in an
76 :obj:`torch_geometric.data.Data` object and returns a boolean
77 value, indicating whether the data object should be included in the
78 final dataset. (default: :obj:`None`)
79 """
