13 class PyGUPFDDataset(InMemoryDataset):
14 r"""The tree-structured fake news propagation graph classification dataset
15 from the `"User Preference-aware Fake News Detection"
16 <https://arxiv.org/abs/2104.12259>`_ paper.
17 It includes two sets of tree-structured fake & real news propagation graphs
18 extracted from Twitter.
19 For a single graph, the root node represents the source news, and leaf
20 nodes represent Twitter users who retweeted the same root news.
21 A user node has an edge to the news node if and only if the user retweeted
22 the root news directly.
23 Two user nodes have an edge if and only if one user retweeted the root news
24 from the other user.
25 Four different node features are encoded using different encoders.
26 Please refer to `GNN-FakeNews
27 <https://github.com/safe-graph/GNN-FakeNews>`_ repo for more details.
28  
29 .. note::
30  
31 For an example of using UPFD, see `examples/upfd.py
32 <https://github.com/rusty1s/pytorch_geometric/blob/master/examples/
33 upfd.py>`_.
34  
35 Args:
36 root (string): Root directory where the dataset should be saved.
37 name (string): The name of the graph set (:obj:`"politifact"`,
38 :obj:`"gossipcop"`).
39 feature (string): The node feature type (:obj:`"profile"`,
40 :obj:`"spacy"`, :obj:`"bert"`, :obj:`"content"`).
41 If set to :obj:`"profile"`, the 10-dimensional node feature
42 is composed of ten Twitter user profile attributes.
43 If set to :obj:`"spacy"`, the 300-dimensional node feature is
44 composed of Twitter user historical tweets encoded by
45 the `spaCy word2vec encoder
46 <https://spacy.io/models/en#en_core_web_lg>`_.
47 If set to :obj:`"bert"`, the 768-dimensional node feature is
48 composed of Twitter user historical tweets encoded by the
49 `bert-as-service <https://github.com/hanxiao/bert-as-service>`_.
50 If set to :obj:`"content"`, the 310-dimensional node feature is
51 composed of a 300-dimensional "spacy" vector plus a
52 10-dimensional "profile" vector.
53 split (string, optional): If :obj:`"train"`, loads the training
54 dataset.
55 If :obj:`"val"`, loads the validation dataset.
56 If :obj:`"test"`, loads the test dataset.
57 (default: :obj:`"train"`)
58 transform (callable, optional): A function/transform that takes in an
59 :obj:`torch_geometric.data.Data` object and returns a transformed
60 version. The data object will be transformed before every access.
61 (default: :obj:`None`)
62 pre_transform (callable, optional): A function/transform that takes in
63 an :obj:`torch_geometric.data.Data` object and returns a
64 transformed version. The data object will be transformed before
65 being saved to disk. (default: :obj:`None`)
66 pre_filter (callable, optional): A function that takes in an
67 :obj:`torch_geometric.data.Data` object and returns a boolean
68 value, indicating whether the data object should be included in the
69 final dataset. (default: :obj:`None`)
70 """
